# 📊 Customer Behavior Analysis

## 📌 Project Overview

This project presents an end-to-end **Customer Behavior Analysis** solution using **Python, SQL, and Power BI**. The objective is to transform raw customer shopping data into meaningful business insights that support data-driven decision-making.

The project focuses on cleaning and analyzing customer shopping data, performing exploratory data analysis (EDA), answering business questions using SQL, and building an interactive Power BI dashboard to visualize customer behavior and sales performance.

---

# 🎯 Project Objectives

- Clean and preprocess raw customer data.
- Perform Exploratory Data Analysis (EDA).
- Analyze customer purchasing behavior using SQL.
- Build an interactive Power BI dashboard.
- Generate actionable business insights.
- Support data-driven business decisions.

# 🚀 How to Run This Project

### 1. Clone the repository

```bash
git clone https://github.com/yourusername/Customer-Behavior-Analysis.git
```

### 2. Install required libraries

```bash
pip install pandas numpy matplotlib seaborn
```

### 3. Open the Jupyter Notebook

```bash
jupyter notebook
```

Run:

```
data_cleaning&EDA.ipynb
```

### 4. Execute SQL Queries

Run:

```
sql_analysis.sql
```

using PostgreSQL.

### 5. Open the Dashboard

Open:

```
customer_behaviour.pbix
```

using **Power BI Desktop**.
